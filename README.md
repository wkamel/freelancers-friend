# Freelancer's Friend
### List of Python offers from sites like freelancer.com, guru.com and others

## How it works?
This script download's Python offers in many formats, i.e.:
freelancer.com - JSON data are used 
guru.com - RSS(in fact Atom) XML  is used 
others ... in progress
